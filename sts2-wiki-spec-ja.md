# Slay the Spire 2 敵・ボス攻略Wiki 仕様書（日本語版）

## 概要
Slay the Spire 2 の敵・ボス攻略に特化した自分用Wikiサイト。
全Act・全敵のデータを日本語で初期登録し、行動パターン・数値データ・自分用メモを管理する。
**サイト全体を日本語で実装する。**

## 技術スタック
- **フレームワーク**: Next.js (App Router)
- **スタイリング**: Tailwind CSS
- **データ管理**: JSON ファイル（`/data/enemies.json`）
- **メモ保存**: localStorage（自分用なのでバックエンド不要）
- **デプロイ**: ローカルで `npm run dev` で使う前提（後からVercel等に置いてもOK）

## デザイン
- **テーマ**: ダークテーマ（ゲームの雰囲気に合わせる）
- **カラー**: 
  - 背景: 深い暗色（#0a0a0f 〜 #1a1a2e 系）
  - アクセント: ゴールド/琥珀（#d4a44a）でカードゲームらしさ
  - テキスト: 明るいグレー（#e0e0e0）
  - 敵タイプ別カラー: ボス=赤系、エリート=紫系、通常=グレー系
- **フォント**: ゲームWiki感のある少しファンタジー寄りのフォント（日本語対応フォント）
- **レイアウト**: サイドバー無し、シンプルなカード型一覧 → 詳細ページ

## ページ構成（すべて日本語UI）

### 1. トップページ `/`
- サイトタイトル「STS2 敵図鑑」
- 検索バー（敵名で日本語インクリメンタルサーチ、英語名でもヒット）
- フィルター:
  - Act別: 第1層 繁茂の地 / 第1層 地下水路 / 第2層 魔窟 / 第3層 栄光の路
  - タイプ別: ボス / エリート / 通常
- 敵一覧（カード型グリッド表示）
  - 各カードに: 日本語敵名 / 英語名（小さく） / Act名 / タイプバッジ / HP

### 2. 敵詳細ページ `/enemies/[slug]`
- **ヘッダー**: 日本語名（大きく）、英語名（小さく）、Act名、タイプバッジ、画像
- **基本ステータス**: HP、出現Act/バイオーム
- **特殊能力**: 固有能力の名前と効果説明（日本語）
- **行動パターン表**: ターン順に行動を表形式で表示
  - 列: ターン / 行動名 / 効果 / ダメージ値
  - フェーズがある敵は「第1フェーズ」「第2フェーズ」等でセクション分け
- **攻略メモ（プリセット）**: 各サイトから収集した攻略情報を日本語で記載
- **自分用メモ欄**:
  - テキストエリア
  - 「保存」ボタンで localStorage に保存
  - 保存済みメモがあれば表示
  - メモの最終更新日時も表示

## 画像の扱い
- wiki.gg の画像URLを参照する形で実装
- 画像が見つからない敵はプレースホルダー画像を表示
- 画像URLは enemies.json の `imageUrl` フィールドで管理

## ボス・エリート 詳細行動パターン（初期データ）

※以下はアセンション無し（初回難易度）のデータ。

### 繁茂の地エリート

**古の偶像（Bygone Effigy）HP127**
- 特殊能力「スロウ」：カードをプレイする度に被ダメ+10%
- 1T: 睡眠 → 2T: 筋力+10 → 3T: ATK25 → 4T: ATK25
- 攻略: 速攻必須。3T目までに倒したい。

**ビャードニス（Byrdonis）HP93**
- 特殊能力「縄張り意識」：毎ターン終了時、筋力+1
- 1T: ATK16 → 2T: ATK4×3 → 3T: ATK18 → 4T: ATK6×3（ループ）
- 攻略: 筋力スケーリングするので早めに倒す。

**寄生カエル（Phrog Parasite）HP61**
- 特殊能力「寄生」：倒されると何かを召喚
- 1T: 「感染」3枚混入 → 2T: ATK4×4 → 3T: 「感染」3枚混入
- 倒すとリグラー4体出現（HP17〜21）。スタン状態で出現。

### 繁茂の地ボス

**ヴァントム（Vantom）HP173**
- 特殊能力「滑る」×9：被ダメージを1に軽減
- 1T: ATK7 → 2T: ATK6×2 → 3T: ATK27＋負傷3枚 → 4T: 筋力+2（ループ）
- 攻略: 多段攻撃でスリップ9回分を消化。長期戦は筋力バフ＋負傷で不利に。

**儀式の獣（Ceremonial Beast）HP252**
- 特殊能力「すき込み」：HP150以下でスタン＆筋力喪失
- 前半: 1T: すき込みバフ → 2T: ATK18＋筋力+2 → 3T: ATK20＋筋力+2 → 4T: ATK22＋筋力+2
- HP150以下後: 1T: 耳鳴りデバフ → 2T: ATK15 → 3T: ATK16＋筋力+3
- 「耳鳴り」：1枚しかカードをプレイできない。3ターン周期。1枚で大量ブロックを出せるカードがあると安心。

**血族の司祭（Kin Priest）HP190**
- トリオ戦（司祭+信者2体、信者HP58〜59）
- 司祭: 1T: ATK8＋脆弱1 → 2T: ATK8＋脱力1 → 3T: ATK3×3 → 4T: 筋力+2
- 信者は筋力バフしながら攻撃。司祭撃破でミニオン逃走。
- 攻略: 基本は信者から処理。全体攻撃があると楽。

### 地下水路エリート

**テラーウツボ（Terror Eel）HP140**
- 特殊能力「悲鳴」：HP70以下で初回スタン
- 1T: ATK17 → 2T: ATK3×3＋活力7 → HP70以下: 弱体99付与
- 攻略: HP70を割ればスタン。その後は弱体で火力激増するのでブロック確保。

**蠢く群生体（Skulking Colony）HP79**
- 特殊能力「硬い殻」：1ターンに20以上のHP減少不可
- 1T: ATK9＋めまい4枚 → 2T: ATK16 → 3T: ブロック10＋筋力+3 → 4T: ATK9×2
- 攻略: 最速4ターンで撃破。安定したダメージソースで削る。

**幻影蟲（Phantasmal Gardener）HP28〜32×複数**
- 特殊能力「臆病」：毎ターン最初に攻撃を受けた時ブロック6獲得
- 複数体が1Tずつずれて行動（多段→ATK5→ATK7→バフの共通順序）
- 攻略: 多段攻撃で臆病のブロックを処理してからダメージ。

### 地下水路ボス

**滝の巨人（Waterfall Giant）HP250**
- 特殊能力「蒸気噴出」：死亡時に蓄積ダメージ（初期15、毎ターン+3）
- 1T: 蒸気噴出15バフ → 2T: ATK15＋脱力1＋蒸気+3 → 3T: ATK10＋蒸気+3 → 4T: HP15回復＋蒸気+3 → 5T: ATK20＋蒸気+3 → 6T: ATK13＋蒸気+3
- 攻略: 倒した後の蒸気噴出ダメージに備えてブロック/HP確保。山札確認して防御札を引けるタイミングで倒す。

**ソウルフィッシュ（Soul Fysh）HP211**
- 1T: 「手招き」を山札1枚＋捨て札1枚混入 → 2T: ATK16 → 3T: ATK7＋手招き1枚 → 4T: 霊体2 → 5T: ATK11＋弱体3
- 「手招き」：手札にあるとターン終了時HP6減少。エナジーに余裕があれば使って排除。
- 攻略: 筋力スケーリングなし。手招き管理が鍵。霊体ターンは攻撃控える。

**ラガヴーリン・メイトリアーク（Lagavulin Matriarch）**
- 睡眠状態で開始。プレート（ターン終了時ブロック12）持ち。
- 3ターン毎に起きて大攻撃＋永続筋力/敏捷デバフ（-2/-2）
- 攻略: 自分のタイミングで起こす。Poison/バーストで削る。

### 魔窟エリート

**万足ムカデ（Decimillipede）複数体**
- 3体同時。倒しても他の生存中は復活。全体攻撃で同時撃破が必要。
- 筋力バフをしている個体から優先撃破。爆発ポーションが有効。

**エントマンサー（Entomancer）HP145**
- 特殊能力「パーソナルハイヴ」：アタック被弾時「めまい」を山札に追加
- 1T: ATK3×7 → 2T: ATK18 → 3T: パーソナルハイヴ+1＋筋力+1（ループ）
- 攻略: スキル主体で戦うとめまい回避可能。多段攻撃は被害増大するので注意。ソブリンブレード等の一撃高火力が有効。

**寄生されたプリズム（Infested Prism）**
- アーティファクト持ち。アタックカードが有効。

### 魔窟ボス

**クラッシャー＆ロケット（Kaiser Crab）**
- 2体の集団戦（合計HP約400）
- 「バックアタック」：背後攻撃時ダメージ50%増
- 「カニの怒り」：味方死亡時、生存個体が筋力5＋99ブロック獲得
- 攻略: 全体攻撃/Poisonで両方同時に削る。片方だけ倒すともう片方が暴走。

**知識の悪魔（Knowledge Demon）**
- 周期的にデバフ選択を強制:
  - 「崩壊」：毎ターンダメージ（6→7→...）
  - 「精神腐敗」：ドロー枚数-1
  - 「怠惰」：カード3枚制限
  - 「枯渇」：エナジー-1
- 4ターン毎にATK＋HP回復＋筋力+2
- 攻略: デバフは全回避だと崩壊ダメージが20超えるので1〜2個は受け入れる。ドロー/エナジー生成手段を確保。

**貪りの獣（The Insatiable）**
- 「蟻地獄」：カウントダウン式即死（初期4ターン）
- 「決死の逃亡」カードで延長可能（使用毎にコスト増加）
- Phase3: 霊体（Intangible）を隔ターン使用
- 攻略: 決死の逃亡は手札に来たら基本使う。ブロックしつつ攻撃する忙しい戦闘。

### 栄光の路エリート

**ナイト三人衆（Knight Trio）**
- フレイルナイト：強打＋連打＋筋力バフ
- マギナイト：2T目「減衰」で全カードのアップグレード解除（撃破で復帰）
- スペクトルナイト：1T目「呪術」で全カードにエセリアル付与（撃破で解除）
- 攻略: スペクトルナイトかマギナイトを先に倒してデバフ解除。

**メカナイト（Mecha Knight）**
- アーティファクト3持ち。「火傷」4枚手札混入→バフ→強打のループ。
- 強打が非常に高火力。バッファーで1に軽減可能。
- 攻略: アーティファクト消費後に脱力が有効。状態異常対策があると長期戦OK。

**ソウルネクサス（Soul Nexus）**
- 弱体/脆弱付与＋攻撃。4連打あり。
- 攻略: Act3エリートの中では比較的易しい。筋力低下カードがあると有利。

### 栄光の路ボス

**ドアメーカー（Doormaker）**
- 1T目：ドア状態（猶予ターン）。2T目から口→目→手の3形態ループ。
- 「口」形態：攻撃。「目」形態：カードドロー不可等のデバフ。「手」形態：1枚目コスト増だが攻撃しない（攻撃チャンス）。
- 攻略: 1T目にパワーカード展開。手形態で削る。自動プレイ系カード/レリックがコスト増を無効化。

**クイーン（Queen）**
- トーチヘッド（ミニオン）との集団戦。
- トーチヘッド存命中はクイーンは攻撃せず、代わりにトーチヘッドの筋力をバフし続ける。
- トーチヘッドの3回目攻撃が連打で高火力。
- 攻略: 全体攻撃でクイーンのHPも並行して削りつつトーチヘッドを処理。

**実験体 #C18（Test Subject）**
- 3フェーズ制（各フェーズでHP削り切りで形態変化。筋力はリセット）
- 第1形態（ボスグレムリン能力）：スキル使用で筋力増。速攻必須。マルチだと体力増で最大の鬼門。
- 第2形態（刺創の本）：防御しきれない分負傷混入。基礎火力高め（毎ターン40程度の攻撃を捌けるブロック必要）。
- 第3形態（ネメシス）：2ターンに1度霊体。安定ブロック＋デッキ回転があれば問題なし。
- 攻略: 第1形態が最難関。全力で速攻。以降は安定したブロックがあれば勝てる。

## 機能詳細

### 検索機能
- 日本語名・英語名どちらでもインクリメンタルサーチ
- 入力中にリアルタイムで一覧をフィルタリング

### フィルター機能
- Act フィルター（複数選択可）: 繁茂の地 / 地下水路 / 魔窟 / 栄光の路
- タイプフィルター（複数選択可）: ボス / エリート / 通常
- フィルターはURLクエリパラメータに反映（ブックマーク可能に）

### メモ機能
- 各敵の詳細ページにテキストエリアを設置
- 「保存」ボタンで localStorage に `memo_{enemyId}` キーで保存
- メモの最終更新日時を表示
- 「メモをクリア」ボタンも設置

## UI テキストはすべて日本語
- ナビゲーション: 「トップ」「敵一覧」
- フィルターラベル: 「Act で絞り込み」「タイプで絞り込み」
- 検索プレースホルダー: 「敵名で検索...」
- ボタン: 「保存」「メモをクリア」「全て表示」
- バッジ: 「ボス」「エリート」「通常」
- 行動パターン表ヘッダー: 「ターン」「行動」「効果」「ダメージ」

## 実装の注意事項
- 通常モンスターのHP・行動パターン詳細は空欄OK（後から自分で埋める）
- ボスとエリートは上記の詳細データを登録した状態で初期化
- 画像URLは空文字で初期登録（後から追加可能）
- レスポンシブ対応（PCメインだがタブレットでも見れるように）
- `enemies.json` を編集すれば敵の追加・更新が簡単にできる構造にする
- 通常モンスターの日本語名はゲーム内テキストと異なる可能性あり。後から修正できるようにする。

## Claude Codeへの指示

4. 敵詳細ページ（ステータス + 行動パターン表 + 攻略メモ + 自分用メモ機能）を実装
5. ダークテーマのスタイリングを適用
6. ローカルで動作確認できるようにする
